Question:

Compare the different products in the Cloud Infrastructure and Platform Services market. (e.g., AWS, Microsoft Azure, GCP, IBM Cloud, Alibaba Cloud, Tencent Cloud, Oracle Cloud Infrastructure (Gen 2), Huawei Cloud, etc.).

Solution:

What is Cloud Infrastructure & Platform Services market?

  The cloud infrastructure and platform services (CIPS) market as standardized, highly automated offerings, in which infrastructure resources (e.g., computer, networking and storage) are complemented by integrated platform services.

  These include managed application, database, and functions as-a-service offerings. The resources are scalable and elastic in near real time and are metered by use.

  Self-service interfaces, including a web-based user interface (UI) and an API, are exposed directly to the customer. The resources may be single-tenant or multitenant and can be hosted by a service provider or on-premises in the customers data center.

Different products in CIPS

1)                           Amazon Web Services

  A subsidiary of Amazon and founded in 2006.

  It is one of the first cloud computing platforms that became widely available.

  According to Canalys, Amazon is an IaaS market leader, holding 31% of the cloud market share.

  AWS has over 175 cloud services for a broad range of use cases and industries.

  AWS fees are based on a pay-as-you-go model, depending on the hardware and software or networking options chosen. 

  AWS has the largest market share in IaaS and PaaS and is a leader in the majority of cloud products provided worldwide.

  Despite proclaimed price reductions, the price for some services, like the AWS computer service, has not become cheaper since 2014. 

  AWS is the most mature and enterprise-ready provider with an influential track record of customer success.

2)                           Microsoft Azure

  Microsoft Azure holds a strong second place after AWS. 

  Canalys reports that Microsoft Azures market share among IaaS cloud providers is 20%.

  Azure is particularly well suited for organizations using Microsoft services. 

  Microsoft support is quite expensive.

  The use cases of Microsoft Azure are diverse and manifold, thanks to the number of service offerings the company provides.

  Microsoft Azure costs like AWS. There are some use cases when Azure is more affordable and when AWS is more economical.

  Microsoft Azure has over 600 services.

3)                           Google Cloud Platform

  Google Cloud has third place on Gartners Magic Quadrant of cloud providers, after AWS and Microsoft Azure.

  The market share of Google Cloud in infrastructure, as a service market, is 7% per Canalys.

  The Google Cloud platform offers 100 products that can be grouped into six categories: storage, databases, computing and hosting, networking, big data, and machine learning.

  Google Cloud offers a pay-as-you-go model. 

  Google Cloud stands out in big data, machine learning and data science capabilities with its products like TensorFlow, ML Kit and Google Datasets.

  Google has challenges with positioning itself as an enterprise-class IaaS solution. Its offerings have not yet reached the level of enterprise maturity that AWS and Microsoft Azure provide.

  Google Cloud customers dont need to be afraid of the vendor lock-in since the main Google offerings are open source.

4)                           Alibaba Cloud

  As per Gartners report, Alibaba Cloud holds a solid fourth place amongst cloud service providers, after AWS, Microsoft Azure and Google Cloud.

  Alibaba Cloud has the biggest market share among cloud providers in China and Asia-Pacific for IaaS and IUS.

  Alibaba Cloud has a lower cost, by around 25%, than overseas cloud service providers.

  Alibaba has limited adoption in other parts of the world, except for South Asia.

  Alibaba Cloud services, such as Express Connect, have a 44% better average network latency than the public ISP between the US and China. 

5)                           Oracle Cloud

  Oracle cloud holds fifth place among cloud providers, after AWS, Microsoft Azure, Google Cloud and Alibaba.

  Oracles Cloud market share is around 2% and the company ranks fifth globally.

  Oracle offers 65 cloud services, including the industry-standards like Kubernetes, Terraform and CloudEvents

  When it comes to payment models, cloud pricing models are complex. The price depends upon different factors, such as CPU and memory usage. 

  Oracle Cloud has a strong partnership with Microsoft Azure so that their customers can run their workloads across the two clouds. 

  Snowflake (data warehousing solution) is not yet available on Oracle.

  Oracle stands out among other cloud providers in its meticulous architecture and its cloud services that are competitive.

6)                           IBM Cloud

  IBM Cloud ranks sixth among the cloud providers.

  IBM holds less than 5% of the cloud market share.

  IBM lists 174 cloud services for AI and ML, analytics, containers, databases, developer tools, IoT, logging and monitoring, networking, storage, and security.

  IBM regularly reviews its cloud product prices. However, the total cost for some scenarios is higher than the ones offered by IBMs rivals.

  IBM is trying to reshape itself as a hybrid cloud and AI company. 

  IBM Cloud revenue growth is 22%, which is inferior to the 45-50% growth rate demonstrated by AWS Cloud and Microsoft Azure.

  IBM Cloud offers 20 always-free cloud products and 11 short-term free trial products.
